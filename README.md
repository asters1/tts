## 微软官网只有试听功能，本程序可以下载音频文件

## 如何使用?

1. 下载源代码使用

```
git clone https://github.com/asters1/tts

cd tts

go run main.go
```

2. 或者直接下载已经编译好的文件

安卓高级终端:
```
https://github.com/asters1/tts/releases/download/v0.1.2/tts_android_termux
```
